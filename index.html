FAQ

Built for trust, clarity, and scale.

Can users damage the app by inspecting the website?

No, not if the real rules are enforced on Firebase and your payment verification runs in Netlify Functions. The frontend should never be trusted for admin access or payment confirmation.

Can the admin upload finished PDF books from a laptop?

Yes. This project is now structured around a PDF-first publishing workflow so the author can upload finished book files instead of typing chapters.
